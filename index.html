Text Treatments

Change the look of your text with CSS rules:

  • font-style - Italic
  • font-variant - Small Caps
  • text-decoration - Underline, line-through
(Duckett, pgs. 280-282)

HTML5 Semantic Markup

Try some markup based on the meaning of your text:

  • <cite> - For Titles of Books and Other Works
  • <abbr> - For Abbreviations and Acronyms
  • <kbd> - For keyboard inputs
  • <code> - For computer code
(Duckett, pgs. 50-58)
